# Minimal Action Card Template

Use this template when the user has reached action convergence.
The card should be treated as a proposed artifact until the user explicitly accepts it.

~~~text
最小行动卡
行动名称：
对应系统：
当前候选瓶颈：
本轮目标：
只做这一件事：
暂缓事项：
完成标准：
负责人：
截止时间：
复盘问题：
~~~

## Field Notes

### 行动名称
A short title for the action.

### 对应系统
The system currently being discussed.

### 当前候选瓶颈
The current candidate bottleneck.
Use “candidate” unless the user has confirmed it.

### 本轮目标
The immediate goal of this action round.

### 只做这一件事
The single action to release now.
This field implements Choke the Release.

### 暂缓事项
Actions that should not be released in this round.
This field prevents over-release.

### 完成标准
The observable condition that allows the user to judge whether the action is complete.

### 负责人
The person responsible for this action.

### 截止时间
The expected completion time.

### 复盘问题
The question to answer after the action is completed.
