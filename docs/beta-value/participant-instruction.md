# Eliy S'FOCUS 闭门体验说明

## 体验入口

请使用浏览器打开：

```text
https://hk-beta.eliyai.com/index.html
```

## 登录步骤

本次体验有两层访问。

第一层是浏览器访问账号，用来进入闭门体验入口。  
第二层是 Eliy 内测登录，用来进入 Eliy 聊天页面。

请使用本轮提供给你的账号信息。不要把账号转发给他人。

## 推荐使用方式

建议使用电脑浏览器，或较新的手机浏览器。进入后，请带一个真实但不涉及敏感信息的问题，与 Eliy 走一轮 S'FOCUS。

你可以这样开始：

```text
我想用 S'FOCUS 澄清一个经营问题。现在问题很多：____。请先帮我定义系统，再帮我找当前候选瓶颈。
```

## 建议带入的问题类型

适合带入：

- 课程交付卡住；
- 老师时间不足；
- 学员复盘跟不上；
- 销售转化低；
- 团队执行分散；
- 多个问题同时出现，不知道先处理哪个。

## 不建议带入的问题类型

请不要输入：

- 公司机密数据；
- 客户隐私；
- 财务敏感数据；
- 个人隐私；
- 需要法律、医疗、投资等专业意见的问题。

如果问题涉及真实业务，请做必要脱敏。你可以描述结构和现象，不需要提交具体客户、金额、合同或个人身份信息。

## 如何判断这次体验是否完成

一次体验完成，通常意味着你已经：

1. 说出了要分析的系统；
2. 说明了这个系统当前目标；
3. 列出了多个当前问题；
4. 和 Eliy 选出了一个当前候选瓶颈；
5. 理解了为什么不要同时处理太多问题；
6. 形成了一张最小行动卡；
7. 判断这张行动卡是否愿意执行。

如果没有走完，也可以填写反馈。卡在哪里本身就是重要信息。

## 如何填写反馈

体验结束后，请填写：

1. 技术通行检查；
2. S'FOCUS 价值反馈；
3. 主体型体验观察；
4. 后续行动复盘报名。

请尽量写具体例子。比如哪句话帮助你看清了问题，哪句话让你觉得空泛或不像真实陪练。

## 如何报名后续复盘

如果你愿意真的执行这张行动卡，可以报名 7 天后的行动复盘。复盘不是考核，而是一起看：这张行动卡是否足够小、是否真的能推动系统变化。
