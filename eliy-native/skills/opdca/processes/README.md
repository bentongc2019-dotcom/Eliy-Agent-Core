# O’PDCA Operating Process Assets

This directory stores operating process assets for the O’PDCA Skill Pack.

**Chinese brand name:** 计划经营

**English system name:** O’PDCA Management System

## Purpose

Operating process assets are scaffolds, not runtime behavior. They describe the
provisional annual operating cycle that the O’PDCA Skill Pack supports.

## Contents

- [`annual-operating-cycle.md`](./annual-operating-cycle.md) — Describes the
  provisional annual operating cycle across nine process areas.

## Boundary

This PR does not implement:

- Workflow execution
- Runtime invocation
- Registry loading
- Retrieval
- Provider behavior
- Persistence

These assets are reference scaffolds only. Actual runtime behavior is owned by
the Runtime Kernel.
