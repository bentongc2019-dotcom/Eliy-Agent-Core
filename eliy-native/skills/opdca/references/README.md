# References — O’PDCA Management System

This directory stores reference assets for the **O’PDCA Skill Pack**.

**O’PDCA Management System** (计划经营) is the real-world management framework this
skill pack helps operators study, practice, and improve.

## Purpose

- Reference assets are **source material scaffolds**, not runtime behavior.
- All references must be mapped through [`source-map.md`](./source-map.md).
- New material intake must follow [`intake-notes.md`](./intake-notes.md).

## Reference Areas

The O’PDCA Skill Pack reference scaffold covers all ten reference areas:

1. Course System Overview
2. Annual Objective Setting
3. Internal Intelligence
4. External Intelligence
5. Annual Plan and Budget
6. Department Objective Alignment
7. Department Operating Plan
8. Review and Improvement
9. Performance, Rewards, Compensation, and Promotion
10. Management Development

## Scope

- This PR does **not** add official corpus content.
- This PR does **not** implement RAG, retrieval, runtime invocation, registry loading,
  or provider behavior.
- This scaffold exists so future PRs can add, review, and map reference material
  without touching runtime code.
